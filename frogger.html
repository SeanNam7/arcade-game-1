Arcade Game: Frogger

Play Now

Overview

This is a personal project in which I recreated the Classic Arcade Game Frogger using HTML5, JavaScript, jQuery, and Object-Oriented Programming. I used HTML5 to add a canvas that consists of grass, rocky floors and a water area, while also adding a player and an infinite amount of bugs to the game to make it more fun as well as challenging. The player is controlled using the keyboard arrows, which is programmed using JavaScript, jQuery, and Object-Oriented programming. In the process of building this game I learned how challenging yet fun it is to build games using programming skills.

Instructions to play game

Your goal is to avoid colliding into bugs, so you can reach the water section and win.

To move your character, use the keyboard arrows. For instance, up keyboard arrow moves up, down keyboard arrow moves down, and so on.

You will have 3 lives and for every time you reach water (win), your score will increase by 100.

I hope you enjoy playing it as much as I enjoyed building it!
